神经网络--线性模型与梯度下降
发表于 2020-03-08| 更新于 2020-03-08|深度学习神经网络

线性回归是监督学习中非常简单的模型，同时梯度下降也是深度学习中应用最广的优化算法。

一元线性回归

假设有变量xix_{i}xi​和目标yiy_iyi​每个，每个i对应一个数据点，希望建立一个模型：

y^i=wxi+b\hat y_i = wx_i+b y^​i​=wxi​+b

y^i\hat y_iy^​i​是我们预测的结果，我们通y^i\hat y_iy^​i​来拟合目标yiy_iyi​，通俗说就是找到这个函数拟合yiy_iyi​使得误差最小，即最小化：

1n∑i=1n(y^i−yi)2\frac {1}{n} \sum_{i=1}^n (\hat y_i-y_i)^2 n1​i=1∑n​(y^​i​−yi​)2

如何最小化这个误差，需要用到梯度下降。

梯度下降法

梯度

梯度就是导数，如果是一个多元函数，那么梯度就是偏导数，比如函数f(x,y)f(x,y)f(x,y)，那么f的梯度就是：

(∇f∇x,∇f∇y)(\frac {\nabla f}{\nabla x},\frac {\nabla f}{\nabla y}) (∇x∇f​,∇y∇f​)

可以称为gradf(x,y)grad f(x,y)gradf(x,y)或者∇f(x,y)\nabla f(x,y)∇f(x,y)。具体某一点(x0,y0)(x_0,y_0)(x0​,y0​)的梯度就是∇f(x0,y0)\nabla f(x_0,y_0)∇f(x0​,y0​).一个点的梯度值就是这个函数变化最快的方向，具体来说，对于函数f(x,y)f(x,y)f(x,y)在点(x0,y0)(x_0,y_0)(x0​,y0​)处沿着梯度∇f(x0,y0)\nabla f(x_0,y_0)∇f(x0​,y0​)的方向，函数增加最快，也就是沿着梯度方向能够很快找到函数的极大值点，通过沿着梯度的反方向能够很快找到函数的最小值点。

梯度下降法

现在需要最小化这个误差，也就是找到这个误差的最小值点，那么沿着梯度反方向就能找到误差的最小值点。我们不断的改变w和b的值，最终找到一组最好的w和b使得误差最小。在更新的时候需要决定每次更新的幅度，这个长度称为学习率，用η\etaη来表示。不同的学习率会导致不同的结果，学习率太大会导致跳动明显，学习率太小，导致下降非常缓慢。

更新公式：

w:=w−η∇f(w,b)∇wb:=b−η∇f(w,b)∇bw: = w-\eta\frac {\nabla f(w,b)}{\nabla w}b: = b-\eta\frac{\nabla f(w,b)}{\nabla b} w:=w−η∇w∇f(w,b)​b:=b−η∇b∇f(w,b)​

多项式回归模型

y^i=wxi+b\hat y_i = wx_i+b y^​i​=wxi​+b

这是关于x的一次多项式，模型比较简单，没办法拟合复杂的模型，可以使用更高层次模型，如：

y^=w0+w1x+w2x2+w3x3+⋯\hat y = w_0+w_1x+w_2x^2+w_3x^3+\cdots y^​=w0​+w1​x+w2​x2+w3​x3+⋯
